# 슬랙 봇 예제

[Hanu 프레임워크](https://github.com/sbstjn/hanu)를 이용한 슬랙 봇 예제

### 준비할 것

- Go가 설치된 환경
- 슬랙 봇 토큰 ([발급받는 방법](./docs/how-to-issue-a-token.md))

### 실행하는 법

API 토큰 값이 `foobar`이라면,

```bash
go run main.go -token 'foobar'
```
